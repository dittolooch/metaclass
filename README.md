# Intro to Metaclass

## Installation


- Clone this repo

    ```sh
    git clone https://github.com/dittolooch/metaclass.git
    cd metaclass
    ```
- Setup virtualenv & install requirements
    ```sh
    pyenv virtualenv metaclass
    pyenv local metaclass
    pip install -r requirements.txt
    ```
- Open jupter notebook
    ```sh
    jupyter notebook
    ```
